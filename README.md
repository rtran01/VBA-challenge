# VBA-challenge
VBA Homework for EMER-VIRT-DATA-PT-03-2021-U-C
 Code for summarizing stock data aranged in columns: <ticker>	<date>	<open>	<high>	<low>	<close>	<vol>
 For the code to work the data has to be previously sorted by <ticker> and then by <date> in ascending order.
  Code will Output Unique Tickers in a spreadsheet, yearly change in ticker price, percentage change.
  Code will also summarize the highest and lowest percentage, highest volume, with the corresponding tickers.
